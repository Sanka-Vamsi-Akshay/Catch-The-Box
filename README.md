# 🎮 Catch The Box

A fast-paced browser game where your goal is to click (catch) the box as many times as possible before time runs out.

---

## 🚀 Live Demo
👉 https://sanka-vamsi-akshay.github.io/Catch-The-Box/

---

## 🕹️ How to Play
- Click **HIT!** to start the game
- A box appears on the screen
- Click the box to score points
- The box moves to a random position every time you click it
- You have **10 seconds** to score as much as possible

## ✨ Features
- 🎯 Random movement system
- ⏱️ Time-based gameplay
- 📊 Score tracking
- 🔁 Retry and Home functionality
- 🎨 Clean and interactive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
